# Student Scoring System using Rule-Based Intelligence

## Problem Statement
Mentoring systems often collect large volumes of student data but lack structured scoring mechanisms to convert this data into actionable insights.

---

## Solution Overview
Designed a rule-based scoring engine to evaluate student performance across multiple dimensions:
- Academic Performance Score (APS)
- Weekly Work Score (WWS)
- Productivity & Task Management Score (PTMS)
- Consistency Rating Score (CRS)
- Student Risk Index (SRI)

---

## Key Features
- Rule-based scoring logic
- Multi-dimensional performance evaluation
- Risk classification system
- Interpretable scoring mechanism

---

## Tech Stack
- Python
- Pandas
- NumPy

---

## ⚙️ Architecture
Raw Student Data → Rule Engine → Score Calculation → Risk Classification → Output Report

---

## Output
- Generated structured scores for each student
- Classified students into performance categories
- Identified high-risk individuals

---

## Use Cases
Academic monitoring systems
Mentorship programs
Early warning systems for student dropout
